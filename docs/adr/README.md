## Format

Chaque ADR suit le template :
1. **Contexte et problème** — Pourquoi cette décision ?
2. **Décision** — Quelle solution a été choisie ?
3. **Conséquences** — Avantages et inconvénients
4. **Alternatives considérées** — Quelles autres options existaient ?
5. **Références** — Documentation, sources

## Statuts possibles

- ✅ **Accepté** : décision validée et implémentée
- 🟡 **Proposé** : en cours de discussion
- ❌ **Rejeté** : alternative écartée
- 🔄 **Supplanté** : remplacé par une autre ADR
